# With Mama Doula

A single-page React site for With Mama Doula.

## Structure

```text
public/
  images/
    about/
    branding/
    contact/
    hero/
src/
  components/
    layout/
    ui/
  content/
    site-content.js
  sections/
    contact-section.jsx
    faq-section.jsx
    hero-section.jsx
    michelle-section.jsx
    offerings-section.jsx
    philosophy-section.jsx
    qualifications-section.jsx
    support-section.jsx
  styles/
    global.css
  app.jsx
  main.jsx
```

## Development

```bash
npm install
npm run dev
```

## Scripts

- `npm run build` creates a production build.
- `npm run lint` checks the codebase.
- `npm run preview` serves the production build locally.
